# Testing & Debug Süreci

## Amaç

Bu modülün amacı, web sitelerinin HTTP güvenlik headerlarını analiz ederek eksik yapılandırmaları tespit etmektir.

## Test Senaryosu

### Gerçek Site Testi

- URL: https://frudotz.com
- Komut:
```bash
cargo run -- headers https://frudotz.com
````

### Sonuç

Tüm güvenlik headerlarının eksik olduğu tespit edilmiştir.

### Çıktı Özeti

* Strict-Transport-Security ❌
* Content-Security-Policy ❌
* X-Frame-Options ❌
* X-Content-Type-Options ❌
* Referrer-Policy ❌
* Permissions-Policy ❌

### Değerlendirme

Bu sonuç, hedef sistemin güvenlik yapılandırmasının zayıf olduğunu göstermektedir. Özellikle CSP ve HSTS gibi kritik headerların eksik olması, XSS ve MITM gibi saldırılara karşı risk oluşturabilir.

## Karşılaşılan Problemler

* Windows ortamında linker (`link.exe`) eksikliği nedeniyle build hatası alındı

## Çözüm

* Visual Studio Build Tools kurulumu yapılarak sorun giderildi

## Sonuç

Test başarıyla tamamlanmış ve sistem doğru şekilde analiz yapmıştır.
