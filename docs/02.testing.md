# Testing

## Amaç

Sistemin CLI, API ve Web GUI katmanlarında doğru ve stabil çalıştığını doğrulamak.

---

## Test Senaryoları

### 1. CLI Test

```bash
cargo run -- headers https://google.com
````

Beklenen:

* Header analizi yapılır
* Skor ve grade hesaplanır
* JSON rapor oluşturulur

---

### 2. Batch Scan Test

```bash
cargo run -- headers targets.txt
```

Beklenen:

* Çoklu site analiz edilir
* Her site için çıktı üretilir

---

### 3. Web GUI Test

```bash
cargo run -- web
```

Tarayıcı:
[http://127.0.0.1:3000](http://127.0.0.1:3000)

Beklenen:

* Yeni tarama yapılabilir
* Raporlar listelenir
* Arama ve filtreleme çalışır
* Grafik ve detaylar görüntülenir

---

## Error Handling Testleri

* Geçersiz URL → hata mesajı gösterilir
* Timeout → sistem çökmez
* Backend hataları → kullanıcıya yansıtılır
