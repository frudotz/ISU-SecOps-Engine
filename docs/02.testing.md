# Testing

## Amaç

Sistemin hem CLI hem Web GUI üzerinden doğru çalıştığını doğrulamak.

---

## Test Senaryoları

### CLI Test

```bash
cargo run -- headers https://frudotz.com
````

Beklenen:

* Header analizi
* Skor üretimi
* JSON kayıt

---

### Web GUI Test

```bash
cargo run -- web
```

Tarayıcı:
[http://127.0.0.1:3000](http://127.0.0.1:3000)

Beklenen:

* URL girip scan yapılabilmesi
* Raporların listelenmesi
* Detay görüntüleme

---

## Test Sonuçları

* CLI düzgün çalışıyor
* JSON kayıt başarılı
* Web GUI üzerinden scan çalışıyor
* API endpoint'ler doğru cevap veriyor

---

## Karşılaşılan Problemler

* URL format hatası (https eksikliği)
* Rust naming warning'leri

---

## Çözümler

* URL normalize edildi
* allow attribute kullanıldı

---

## Sonuç

Sistem tüm senaryolarda stabil çalışmaktadır.
