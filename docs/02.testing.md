# Testing & Debug Süreci

## Amaç

Bu modülün amacı, web sitelerinin HTTP güvenlik headerlarını analiz ederek eksik yapılandırmaları ve olası güvenlik risklerini tespit etmektir.

---

## Test Senaryoları

### 1. Gerçek Site Testi

- URL: https://frudotz.com
- Komut:
```bash
cargo run -- headers https://frudotz.com
````

### Beklenen Davranış

* Header analizi yapılmalı
* Eksik header'lar listelenmeli
* Severity bilgisi gösterilmeli
* Risk açıklamaları eklenmeli
* Skor ve grade hesaplanmalı
* JSON çıktısı oluşturulmalı

---

## Test Sonucu

* Tüm güvenlik headerlarının eksik olduğu tespit edildi
* Sistem tarafından düşük skor ve F notu verildi
* Her header için risk açıklaması doğru şekilde üretildi
* JSON raporu başarıyla oluşturuldu

---

## Çıktı Özeti

* HIGH seviyede eksikler → CSP, HSTS
* MEDIUM → X-Frame-Options
* LOW → diğer header'lar

Bu dağılım doğrultusunda sistem skor düşürme mekanizmasını doğru şekilde uygulamıştır.
