# Core Architecture

Bu modül, HTTP header analizini gerçekleştiren ana bileşendir.

---

## Sistem Akışı

1. CLI veya Web GUI üzerinden URL alınır
2. Backend analiz fonksiyonunu tetikler
3. HTTP request gönderilir
4. Header'lar kontrol edilir
5. Severity ve risk hesaplanır
6. Skor oluşturulur
7. Sonuç:
   - JSON olarak kaydedilir
   - API üzerinden sunulur
   - CLI'da gösterilir

---

## Bileşenler

### CLI
Kullanıcıdan komut alır ve analiz başlatır.

### Analyzer
Header kontrolünü ve scoring işlemini yapar.

### JSON Storage
Sonuçları `assets/reports/` klasörüne kaydeder.

### Web Server
Axum tabanlı API sağlar:
- /reports
- /scan

### Web GUI
Frontend üzerinden kullanıcı etkileşimini sağlar.
