# Core Architecture

Bu modül HTTP header analizine odaklanır.

## Akış

1. Kullanıcı CLI ile URL girer
2. HTTP request atılır
3. Headerlar parse edilir
4. Eksik/güvenli headerlar belirlenir
5. Sonuç JSON/Markdown olarak üretilir

## Kullanılan Teknolojiler

- Rust
- reqwest
- tokio
- clap
