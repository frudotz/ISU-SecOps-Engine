# Core Architecture

Bu modül HTTP header analizine odaklanır.

## Akış

1. Kullanıcı CLI ile URL girer
2. HTTP request atılır
3. Headerlar parse edilir
4. Eksik/güvenli headerlar belirlenir
5. Sonuç JSON/Markdown olarak üretilir

## Kullanılan Teknolojiler

- Rust
- reqwest
- tokio
- clap

## Güncelleme Notu (v0.2.0)

Sistem artık analiz sonuçlarını yalnızca terminalde göstermekle kalmayıp, aynı zamanda JSON formatında dosyaya kaydedebilmektedir. Bu sayede sonuçlar daha sonra işlenebilir ve analiz edilebilir hale gelmiştir.
