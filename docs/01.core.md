# Core Architecture

Bu modül, HTTP Header Analyzer sisteminin ana analiz motorunu ve bileşenlerini tanımlar.

---

## Sistem Mimarisi

Proje 3 ana katmandan oluşur:

### 1. CLI Katmanı
- Komut satırı üzerinden çalışır
- Kullanıcıdan hedef URL alır
- Batch scan desteği sağlar

### 2. Analyzer Katmanı
- HTTP isteği gönderir
- Header’ları analiz eder
- Severity ve risk hesaplar
- Skor ve grade üretir

### 3. Web Katmanı (Axum)
- REST API sağlar
- Web GUI ile haberleşir
- JSON raporları sunar

---

## Veri Akışı

1. Kullanıcı URL girer
2. HTTP request gönderilir
3. Header'lar alınır
4. Analiz yapılır:
   - Eksik header kontrolü
   - Value analizi
5. Skor hesaplanır
6. Sonuç:
   - CLI çıktısı
   - JSON kayıt
   - Web GUI

---

## Veri Yapıları

### headerResult
- name
- present
- severity
- risk

### report
- url
- grade
- score
- label (Secure / Risky)
- headers

---

## Özellikler

- Async HTTP istekleri (reqwest + tokio)
- Timeout kontrolü
- JSON serialization (serde)
- Web server (axum)
- Frontend dashboard (HTML + JS)
