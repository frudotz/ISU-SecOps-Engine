# Core Architecture

Bu modül, HTTP header analizini gerçekleştiren ana bileşendir ve hem CLI hem Web GUI üzerinden çalışacak şekilde tasarlanmıştır.

---

## Sistem Akışı

1. Kullanıcı CLI veya Web GUI üzerinden URL girer
2. Backend analiz fonksiyonunu tetikler
3. HTTP isteği gönderilir
4. Header'lar kontrol edilir
5. Severity ve risk hesaplanır
6. Skor oluşturulur
7. Sonuç:
   - JSON olarak kaydedilir
   - API üzerinden sunulur
   - CLI'da gösterilir

---

## Bileşenler

### CLI
Komut satırı üzerinden analiz başlatır.

### Analyzer
Header kontrolü, value analizi ve scoring işlemlerini yapar.

### JSON Storage
Sonuçları `assets/reports/` klasöründe saklar.

### Web Server (Axum)
API endpoint'leri sağlar:
- GET /reports
- POST /scan

### Web GUI
Frontend üzerinden kullanıcı etkileşimini sağlar.

---

## Gelişmiş Özellikler

- Header value analizi
- Batch scan desteği
- Top risk summary üretimi
- Timeout yönetimi
